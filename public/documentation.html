Movie API

This is the skeleton for my Movie API, which will have the following features
  • Contain information about movies, directors and genres
  • Allow users to create a profile in order to favourite movies

Allow users to add a movie to their list of favourites /users/[username]/addfavourite/[movietitle] POST None a text message indicating a movie has been added to favourites
Allow users to remove a movie from their list of favourites /users/[username]/removefavourite/[movietitle] DELETE None a text message indicating a movie has been removed from favourites
Allow exiting users to deregister /users/:username DELETE None a text message showing a user has deregistered
